PDORM

Simple Data Mapper ORM for PHP

PDORM is a simple Data Mapper ORM build just for fun. Other ORM's are powerful and more sophisticated but if you need something simple with little configuration this might be the option for you.

As of right now it only supports CRUD operation on a single table, no relationships, no value objects... but, PR's are welcome to add this and other functionalities
