Getting started

Right now it has only been tested on sqlite and mysql but since it is built on top of PDO it should work with more Databases.

Usage

Assuming you have a DB table called plans, create a mapper:

<?php

namespace App\Mepper;

use Enphpity\Pdorm\Mapper;

final class PlanMapper extends Mapper
{
    protected $table = 'plans';
}

Then create a plain class (yes I wanted them to be plain PHP classes):

<?php

namespace App\Domain\Model;

class Plan
{
    protected $id;

    public $name;

    public $price;

    public function getId()
    {
        return $this->id;
    }

    ...
    // More Getters and Setters and other business logic
}

Instaiate the Manager to manage the mappers and add a PDO Connection:


use App\Domain\Model\Plan;
use App\Mepper\PlanMapper;
use Enphpity\Pdorm\Manager;
use PDO;

$manager = new Manager();

// or

$manager = Manager::getInstance();

$pdo = new PDO('someDSN', 'user', 'pass', ['yada' => 'yada']);

$manager->addConnection($pdo);

$manager->addMapper(Plan::class, new PlanMapper(Plan::class));

Finally use it in some controller or service or whatever yoy want to use it:

$planMapper = $manager->getMapper(Plan::class);

$plans = $planMapper->all();

Happy coding!
